# Campus Marketplace Bot

A minimal Telegram bot built with TypeScript and Telegraf.

## Setup

1. Create a bot token with BotFather.
2. Copy the example environment file and add your token:
   - Create a .env file in the project root based on .env.example

## Install

```bash
npm install
```

## Run (development)

```bash
npm run dev
```

## Build and run (production)

```bash
npm run build
npm start
```
